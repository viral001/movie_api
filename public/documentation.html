Documentation

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of ALL movies to the user /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[title] GET None A JSON object holding data about a single movie, containing a title, description, genre property, director property and image. Example:

{ title : 'Iron Man', description : 'After being held captive in an Afghan cave, billionaire engineer \ Tony Stark creates a unique weaponized suit of armor to fight evil.', genre : 'Action', director : 'Jon Favreau', image : 'iron_man.png' }
Get data about a movie genre /genres/[name] GET None A JSON object holding data about a movie genre. Example:

{ name : 'Comedy', description : 'A comedy film is a genre of film in which the main emphasis is on humour. These films are designed to make the audience laugh through amusement and most often work by exaggerating characteristics for humorous effect.' }

Get data about a director /directors/[name] GET None A JSON object holding data about a director. Example:

{ name : 'Greg Mottola', born : 'July 11, 1964' }

Add new user accounts /users POST A JSON object holding data about the user to add, structured like:

{ username : 'janedoe123', password : 'password123', email : 'example@gmail.com', date_of_birth : "January 1, 1990" }

A JSON object holding data about the user that was added, including an ID:

{ id : 1, username : 'janedoe123', password : 'password123', email : 'example@gmail.com', date_of_birth : "January 1, 1990" }

Update a user's information /users/[username] PUT A JSON object holding data about the user to add, structured like:

{ username : 'janedoe123', password : 'password123', email : 'example@gmail.com', date_of_birth : "January 1, 1990" }

A text message indicating the user's information was updated
Add movies to user's list of favorites /users/[username]/favorites POST None A message indicating the movie was added to the user's list of favorites
Remove movies from user's list of favorites /users/[username]/favorites DELETE None A message indicating the movie was removed from the user's list of favorites
Remove user accounts /users/[username] DELETE None A message indicating the user's account was deleted
