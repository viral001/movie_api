Watchr documentation

This web app is designed to provide users with access to information about different movies, directors, and genres.

Users will be able to sign up, update their personal information, and compile a list of their favorite movies.

Request URL HTTP method Query parameters Request body data format Response data format
Return a list of all movies /movies GET None None JSON object holding data about all movies.
Return data (description, genre, director, image URL) about a single movie by title /movies/[name] GET None None JSON object showing movie data including genre, name, and description. Example:
{"Genre":{"Name":"Comedy","Description":"The main purpose of the comedy genre is to amuse the audience."},"Director":{"Name":"Chris Rock","Bio":"Chris Rock is an American comedian, director, producer, and screenwriter.","Birth":"1965-02-07T00:00:00.000Z"},"_id":"5e1110e8c6b8972a9a43264e","Title":"Head of State","Description":"Chris Rock plays Mays Gilliam, a Washington, D.C., alderman who becomes a longshot presidential candidate.","ImagePath":"headofstate.png","Featured":true}
Return data about a genre (description) by name/title /movies/genres/[name] GET None None A text message showing the genre's name and description. Example:

Western

Westerns tell stories primarily in the second half of the 19th century in the American Old West. The protagonists are often cowboys or gunfighters.
Return data about a director (bio, birth year) by name /movies/directors/[name] GET None None JSON object holding data about the director. Example:
{"Name":"Mike Newell","Bio":"Mike Newell is a British director and producer.","Birth":"1942-03-28T00:00:00.000Z"}
Allow new users to register /users POST None JSON object that contains new user data.
Example:
{ "Username": "janeanderson", "Password": "12345", "Email": "janeanderson@gmail.com", "Birthday": "1/1/2001" } 		JSON object showing new user data, including username, password, email, and date of birth
Allow users to update their user info /users/[UserID] PUT None JSON object. Example:
{ "Username": "johnsmith", "Password": "12345", "Email": "johnsmith@gmail.com", "Birthday": "1/1/2001" }		 A text message confirming the user info was updated
Allow users to add a movie to their list of favorites /users/[username]/favorites/[MovieID] POST None None JSON object showing submitted movie data
Allow users to remove a movie from their list of favorites /users/[username]/favorites/[MovieID] DELETE None None A text message confirming the movie was removed
Allow existing users to deregister /users/[UserID] DELETE None None A text message confirming the account was deleted. Example:
janie123 was deleted.
